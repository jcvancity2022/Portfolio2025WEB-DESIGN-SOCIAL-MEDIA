Arthur Erickson Website

This is the Arthur Erickson website project from IAT 235: Information Design created by Jeffrey Cheng.

Project Description

The Arthur Erickson website is a focused showcase of the legacy of a visionary architect. The project began with research into Erickson’s work to understand the philosophy behind his designs. The site was envisioned as a digital exhibition, highlighting projects through concise narrative and imagery.

This was a school project from IAT 235: Information Design at SFU. I was tasked to create a website about a famous architect. I chose Arthur Erickson for his influence and to deepen my skills beyond company sites at the time. I added content covering the architect’s introduction, projects, and design philosophy using HTML/CSS.

Introductory section using a minimalist visual language inspired by Erickson’s architecture. Clean typography and balanced layout echo his emphasis on spatial harmony.

Process Analysis Part 1

Developing the site called for a minimalist design mirroring Erickson’s modern, nature-centric approach. Using Figma, I established a clean layout with generous whitespace so the work remained front and centre. Interactive galleries were implemented with HTML5 and CSS3, reflecting the sophistication of Erickson’s designs.

Screenshot of the homepage layout with interactive elements. Whitespace and structure guide attention—echoing Erickson’s spatial design principles.

Process Analysis Part 2

I began by adding the introduction, projects, and design philosophy in Figma, then translated the layout to code. I included a persistent navigation bar for easy movement across pages. I also added contact, date, and location details plus admission and ticket information—organized within a consistent template for clarity.

Early development stage in Figma. Placeholder content was replaced with real exhibition data, integrating biographical, project, and ticketing info in a unified structure.

Problem 1: Information Overload and Layout Confusion

Initially, all content—biography, architectural projects, and design philosophy—appeared on a single page. This created clutter and cognitive overload, since users expect different content types to be segmented across distinct pages.

Draft homepage before reorganization. Combining everything on one page made navigation difficult and increased mental load.

Solution:

The content was reorganized into separate pages. By segmenting the introduction, detailed projects, events, and contact info, the site’s navigation became clearer and more user-friendly. This aligns with UX best practices for readability and flow.

Revised structure with content distributed across multiple pages, improving clarity and reducing confusion.

Problem 2: Need for a User-Focused and Minimalist Design

The challenge was crafting a design that reflected Erickson’s modern, nature-integrated style. The original single-template approach lacked variation and failed to engage users effectively.

Initial site built on a single, rigid template. Limited variation didn’t capture the diversity and elegance of Erickson’s portfolio.

Solution:

A minimalist design was adopted to mirror Erickson’s aesthetic. Using a clean layout with ample whitespace and interactive galleries placed the projects at the forefront. HTML5 and CSS3 powered subtle interactions for a seamless user experience.

Final layout applying minimalist principles. Grids, soft contrasts, and image-led galleries put Erickson’s work at the visual and conceptual forefront.

Reflection on Solutions:

Reflecting on this portfolio, user experience proved paramount. Moving to a minimalist design improved clarity, while interactive elements deepened engagement with Erickson’s work. Introducing a clear navigation bar and distributing content across pages resolved the initial interface issues.

This project underscored the value of iterative design and continuous refinement. The result is a more intuitive and educational platform where each project receives focused attention—demonstrating the impact of user-focused design in creating compelling digital experiences.
