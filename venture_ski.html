Venture Ski Website

This is the Venture Ski Website by Jeffrey Cheng and Tiger Yang.

Project Description

This was a group project from IAT 339: Web Design and Development at SFU. For the Peak Value Ski Agency, we were tasked with creating a site that captured the excitement of skiing while offering an efficient booking system. We began with user research on ski enthusiasts and competitive sites. Through collaborative ideation, we developed a concept balancing vivid imagery with user-friendly navigation.

The top image shows user personas emphasizing affordable, high-quality trips. Below, a blog post “5 Affordable Ski Resorts in Canada” reflects competitive analysis research.

Process Analysis

I implemented core website components using HTML and enhanced them with JavaScript.
My contributions included:
- Building an engaging company introduction and services page.
- Developing a booking system with intuitive date selection and package customization.
- Using parallax scrolling and high-resolution images to emphasize immersion and visual appeal. The goal was to create a visually rich yet accessible user journey.

Clean interface design with company info, service offerings, and resort listings—showing ratings, amenities, and pricing.

Problem 1: Style Mismatch

The initial design did not fully align with the thematic identity of a ski agency, leading to weak brand resonance with users.

Outdated homepage layout with inconsistent design and event cards lacking thematic cohesion.

Solution:

I conducted additional design research to realign the site’s visuals with the skiing theme. By integrating modern UI concepts in Figma and adjusting imagery, the final design better reflected the brand identity, ensuring stronger user engagement.

Figma mockups showcasing modern, winter-themed UI elements that redefined the website’s visual identity.

Problem 2: Payment Integration Challenge

The integration of a payment system in the booking flow was difficult but crucial for user convenience and streamlined transactions.

Hotel booking interface showing early prototypes and layout constraints.

Solution:

The account page was redesigned to incorporate secure payment functionalities, ensuring users could manage bookings efficiently. This not only enhanced usability but also improved user trust by enabling a smooth and safe transaction process.

Final account page design with improved booking history, payment integration, and overall user flow.

Reflection on Solutions:

The late realization of a style mismatch reinforced the importance of validating design decisions early in the process. Solving the payment integration challenge improved overall usability and highlighted the need for user-focused design in functional components. This project taught me to balance aesthetics, functionality, and accessibility while collaborating effectively in a team.
