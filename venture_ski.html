Venture Ski Website

This is the Venture Ski Wesbite by Jeffrey Cheng and Tiger Yang.

Project Description

This was a group project from IAT 339 which was Web Design and Development from SFU and we have completed this project in IAT 339 Group Project. For the Peak Value Ski Agency, we were tasked to design a website that could encapsulate the exhilarating experience of skiing while providing an efficient booking system. We kicked off the project by researching the target audience—ski enthusiasts—and the competitive landscape. Collaborative ideation led to a concept that balanced vivid imagery with user-friendly navigation.

The image presents two user personas who value affordable, high-quality ski trips. Below, a blog post titled “5 Of The Most Affordable Ski Resorts in Canada” showcases budget-friendly ski vacation options.

Process Analysis

I was responsible for the implementation of core website components using HTML. My efforts were centred on crafting a compelling introduction to the company, detailing the services provided, and embedding a booking system. The booking system was particularly crucial, designed to be intuitive and user-friendly, leveraging JavaScript to facilitate seamless navigation through date selection and package customization. High-definition images were strategically placed throughout the site to visually convey the thrill of skiing, enhanced by parallax scrolling effects to add a sense of depth and immersion.

Clean interface showcasing company info, offerings, and mission—plus resort listings with ratings, amenities, and pricing.

Problem 1: Style Mismatch

The initial design phase did not adequately capture the thematic essence of a ski agency, leading to a potential disconnection with the target audience.

Screenshot of an outdated ski agency homepage with a group of skiers and event cards that lack thematic cohesion.

Solution:

The realization of this mismatch prompted a strategic pivot in the design approach. I undertook additional research to identify a web design style that would more accurately reflect the skiing theme. This critical adjustment, although realized later in the development process, was essential in realigning the website’s aesthetic with its thematic goals, thereby ensuring a more authentic and engaging user experience.

The Figma mockups showcasing modern, winter-themed UI concepts are used to realign the site’s visual identity.

Problem 2: Payment Integration Challenge

A significant challenge identified was the integration of a payment system within the account page, crucial for streamlining the booking process and enhancing user convenience.

The hotel listing (Pangea Pod Hotel) highlights the early booking features and layout limitations.

Solution:

To address this, the account page was specifically developed to incorporate payment functionalities. This development was aimed at providing a straightforward and secure payment process, allowing users to easily manage bookings and transactions. This solution not only improved the website's functionality but also significantly enhanced the user experience by providing a hassle-free payment process.

Redesigned account page showing booking history, payment info, preferences, and improved usability.

Reflection on Solutions:

The late realization of the need for a thematic realignment of the website’s design underscored the importance of early and continuous thematic validation throughout the development process. This experience highlighted the necessity of aligning every design element with the project's thematic and functional goals from the outset. Addressing the payment integration challenge by enhancing the account page functionality marked a pivotal improvement in the website's usability. This adjustment not only solved a critical issue but also set a precedent for prioritizing user convenience and security in transactional functionalities. In reflecting on these solutions, it's evident that addressing both aesthetic and functional challenges was crucial in developing a website that not only meets the users' expectations but also provides an engaging and seamless experience. The insights gained from this project underscore the importance of flexibility, user-centric design, and the continuous evolution of website functionalities to meet and exceed user needs.
